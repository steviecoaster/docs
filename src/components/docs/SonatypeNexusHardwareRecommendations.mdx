To install Sonatype Nexus, your server must meet the following requirements:

- Windows Server 2019 or later
- 4 cores
- 16 GB RAM
- 500 GB disk space for package storage. We recommend using a separate disk for this purpose so that the storage can be easily expanded in the future if needed.
