To begin the installation, first install Chocolatey CLI. This can be done using the following from an elevated PowerShell session

```PowerShell
Set-ExecutionPolicy Bypass -Scope Process -Force
Invoke-RestMethod https://ch0.co/go | Invoke-Expression
```
